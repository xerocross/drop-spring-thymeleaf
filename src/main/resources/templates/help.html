This is an app created by Adam Cross mainly for practice learning Spring boot and related technologies.

What does it do?

You can store random bits of text data and easily tag them by subject using #hashtags. The query feature allows you to easily search all of your existing notes by hashtag.

So the entire purpose of this app is just to store and retrieve bits of text data?

Yes. It's worth noting how querying works though. If you query your notes for "#apple #pear" then it will return only the notes with both the #apple and the #pear hashtags.
