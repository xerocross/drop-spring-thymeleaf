Drop App

This is an app created by Adam Cross mainly for practice learning Spring boot and related technologies.

POST '/drop'

Post text to the /drop endpoint using the query parameter 'dropText'. We call these bits of text data "drops". Text can include #hashtags. These hashtags are used in the internal business logic for querying drops.

GET '/drops'

query

Pass in the 'query' parameter to get a list of drops. The query string can be any string, but the real magic lies in how Drops handles hashtags in the query string. If your query string contains a hashtag, like #apple, then Drop will return the drops that have that hashtag. It will do that for as many hashtags as you include in your string. Thus querying "#apple #pear" will return all of the drops containing both '#apple' and '#pear'. When querying, string that is not part of a hashtag will be ignored.
