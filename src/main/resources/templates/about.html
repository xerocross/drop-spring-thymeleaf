About

This is an app created by Adam Cross mainly for practice learning Spring boot and related technologies.

This app is built on Spring boot. It uses Thymeleaf templates, and it uses Spring Security. It is hosted on Heroku using a MySQL database to persist user data.

What does the app do?

You can store bits of text data and easily tag them by subject using #hashtags. The query feature allows you to easily search all of your existing notes to find them again later.

So the entire purpose of this app is just to store and retrieve bits of text data?

Yes. It's worth noting how querying works though. If you query your notes for "#apple #pear" then it will return only the notes with both the #apple and the #pear hashtags.

After some user feedback, we have made search by hashtag optional. If your search query includes a hashtag, then Drop will use hashtag search. Otherwise, it will revert to a simpler form of search based on simple string containment.

How secure is this app?

Drop should not be used for confidential information. Your notes and password are stored in the database as plain text. Anyone with access to the database can read all of the information. That said, the security should hold up against casual hacking attempts by unauthorized users, depending on the quality of your password.

I do not recommend using this app to store passwords.
